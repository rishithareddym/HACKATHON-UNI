# Zenith Institute of Technology – University Portal

This project is a modern university web portal designed to provide an interactive and user-friendly experience for students, applicants, and visitors. The goal was to move away from traditional static university websites and create something more engaging, intuitive, and informative.

The platform brings together academics, admissions, campus life, and career guidance into a single, cohesive system.



## Overview

The Zenith Institute of Technology portal is built with a focus on usability and structured navigation. It allows users to explore courses, understand career paths, view campus activities, and apply for admissions in a smooth and guided manner.

The design emphasizes clarity, responsiveness, and subtle interactivity without overwhelming the user.


## Features

The portal includes a wide range of features that simulate a real university experience:

- An academics section that provides information about available courses and programs
- A smart course recommendation system that suggests suitable courses based on user interests through an interactive quiz
- A career path visualizer that helps users understand future opportunities and job roles for different streams
- A dedicated alumni section showcasing feedback, experiences, and career journeys
- Sliding panels for clubs and campus activities to highlight student life
- A section for upcoming events with registration functionality
- An admissions page where users can apply for programs
- Interactive and virtual campus maps for better exploration
- Placement insights including branch-wise statistics and highest packages
- Latest news and research updates related to the institution
- A custom animated cursor and smooth UI interactions to enhance user experience



## Design Approach

The interface is designed to be modern and minimal while still being engaging. Instead of adding heavy animations, the focus is on subtle transitions, structured layouts, and consistent styling.

The aim was to create a balance between functionality and visual appeal so that users can navigate easily while still enjoying the experience.



## Tech Stack

The project is built using:

- React with Vite for frontend development
- React Router for navigation
- CSS for styling and layout



## Running the Project

To run this project locally:

1. Clone the repository  
   git clone  https://github.com/rishithareddym/HACKATHON-UNI.git

2. Install dependencies  
   npm install  

3. Start the development server  
   npm run dev  

4. Open in browser  
   http://localhost:5173
